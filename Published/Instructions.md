A program Futtatása:

1. Megnyitjuk a program mappáját visual studio code-ban
2. A terminál fülre kattintva megnyitunk egy új terminált
3. A terminálból bemegyünk a backend mappába "cd backend" parancsal
4. Letöltjük a node module-okat "npm install" parancsal
5. Elindítjuk a backendet "npm start" parancsal
6. Visszamegyünk a gyökérmappába "cd .." parancsal
7. A terminálból bemegyünk a frontend mappába "cd frontend" parancsal
8. Letöltjük a package-eket "npm install" parancsal
9. Elkészítjük a next.js build-et "npm run build" parancsal
10. Elindítjuk a frontend-et "npm start" parancsal
11. Végül ctrl bal click-el megnyitjuk az oldalt a [Local:](http://localhost:3000)-ra kattintva vagy a böngészőbe beírjuk a [Local:](...) zárójelben lévő végpontot
